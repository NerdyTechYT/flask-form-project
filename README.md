## How to Run
1. Ensure Python is installed.
2. Install Flask:
```bash
pip install flask
````

3. Run the app:

```bash
python app.py
```

4. Open your browser at `http://127.0.0.1:5000/`
5. Fill out the form. Submissions are saved automatically in `submissions.txt`.
